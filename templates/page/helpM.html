Руководство пользователя

Функционал приложения делится на две части:

  1. Отбор пользователей для рассылки.
  2. Выполнение рассылки.

Детально:

  1. Закладка Пользователи. По нажатию на кнопку "Отобрать" будут отобраны подписчики, у которых дата последней рассылки <= полю "дата последней рассылки". Если включить флаг "все пользователи", то будет выполнен отбор по всем подписчикам, без учета даты. Заполненный список означает, что либо рассылка еще не была выполнена, либо по каким-либо причинам не была завершена в прошлый раз.
  2. Закладка Отправка. Для выполнения необходимо, чтобы в списке подписчиков была хоть одна запись. В полях "дата с" и "дата по" указывается период публикации статей.

Рассылка писем проходит при помощи Cron. За один раз отправится максимум писем, согласно лимиту хостинга и лимиту, установленному в настройках Рассыльщика. Лимит рассыльщика может установить только администратор СУБД.
